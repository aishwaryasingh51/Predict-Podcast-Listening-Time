üîç Approach Overview#

  1. Exploration & Profiling: Types, missingness, cardinality, drift checks.
  2. Feature Handling: Drop ID, median imputation (numeric), mode imputation (categorical), one‚Äëhot encode.
  3. Baseline: Mean target predictor.
  4. Model Suite: Decision Tree (manual grid), Random Forest (focused grid), Histogram Gradient Boosting (iter/depth/lr sweep).
  5. Selection: Choose model minimizing validation RMSE.
  6. Retrain: Full training data with tuned parameters (disable early stopping for final fit).
  7. Inference: Column alignment + submission generation.

üìä Key Insights#

Target Shape

Moderately dispersed; extreme values limited, enabling stable tree splits.

Correlations

A small cluster of numeric features dominate linear association strength; categoricals vary widely in cardinality.

Drift Check

Train vs test numeric distributions largely aligned after clipping extreme quantiles.

Model Lift

Histogram Gradient Boosting more than halves baseline RMSE on validation split.

üìê Evaluation Metric#

RMSE penalizes large errors quadratically:

$$\mathrm{RMSE} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (y_i - \hat{y}_i)^2}$$

Lower is better. Validation used an 80/20 split (random_state=51) for comparability.
